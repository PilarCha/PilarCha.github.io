Tab Tracker

Save Guitar Tabs with Search

User signs in to add Guitar tabs. Each song is validated to contain Title, Lyrics, Tab, Youtube Link, Artist, Genre, Album. Viewing the Song shows all the attributes.

Users could edit the songs already created. A search bar searches in real time.

View Github

Front-End: Vuetify, Vue.js

Back-End: Javascript ES6, Express.js

Database: Sequilize

NPM Installs: Debounce, JWT Tokens
